# FinOps

Cloud cost management: reviews, tagging, budgets, commitments.

What belongs here:

- Monthly cost reviews: `YYYY-MM-cost-review.md` with the top movers,
  anomalies, and actions taken.
- Tagging policy: the canonical note describing required tags and the
  allocation model.
- Budget and alert inventory: what alerts exist, thresholds, who is notified.
- Commitment tracking: reserved instances / savings plans, coverage, renewal dates.

Suggested frontmatter: `type: cost-review | policy | budget | commitment`.
