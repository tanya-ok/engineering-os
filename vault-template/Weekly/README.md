# Weekly

Weekly review notes: `YYYY-Www.md` (ISO week), one per week.

Suggested structure:

- What moved: shipped work per domain.
- What is blocked: blockers with owners.
- Open loops delta: loops opened, loops closed (sync with `_Index/Open Loops.md`).
- Next week: the two or three things that matter.

An agent can draft this from the week's notes and activity; you finalize it.
