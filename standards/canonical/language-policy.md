# Language policy

- **Written artifacts are in one language.** Code, comments, commit messages,
  documentation, and anything another person might read use a single agreed
  language (English by default). Set yours in the user vault
  (`communication/`).
- **Conversation follows the user.** Talking with the user happens in whatever
  language they choose; the artifacts that outlive the conversation do not.
- **Punctuation and tone conventions** belong in the user vault, not here, so
  they travel with the person rather than the governance layer.
